# ⭐ EAIM Master Class
### AI-based Intelligent Music Ensemble Platform

**Emotion, Artscape, Interactive, and Musicalization**

본 프로젝트는 35년 경력의 음악 교육 전문가와 AI의 협업으로 탄생한 **지능형 합주 가이드 플랫폼**입니다. 악보라는 장벽을 넘어, 누구나 1시간 만에 원곡과 합주할 수 있는 세상을 꿈꿉니다.

## 🚀 Key Features
* **AI Track Separation:** LALAL.AI API를 활용한 원곡 트랙 분리 (Bass, Drums, Other).
* **Transposing Instrument Guide:** Bb, Eb, F 등 모든 이조악기 실시간 핑거링 가이드.
* **Color-Coded Piano:** 음악 언어를 시각적 색상으로 변환한 직관적 건반 가이드.
* **Hangeul Note Sheet:** 초보자를 위한 한글 계이름 악보 자동 생성 및 인쇄.

## 📘 Vibe Coding Journey
이 앱은 전통적인 코딩 방식이 아닌, AI와 음악적 영감을 나누는 **'바이브 코딩(Vibe Coding)'**을 통해 제작되었습니다. 교육 현장의 결핍을 기술로 채워가는 과정을 담은 가이드북의 실전 결과물입니다.
