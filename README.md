## 🎸 EAIM Guitar

원곡을 업로드하면 트랙을 분리하고, 악기별 맞춤 코드를 자동으로 생성합니다.

### 주요 기능
- **AI 트랙 분리** — 원곡에서 베이스/건반/기타/드럼 개별 분리 (Spleeter/Demucs API)
- **멀티파트 연습 모드** — 원하는 파트를 끄고 나머지 원곡과 직접 합주
- **마이너스원 시스템** — 건반+드럼 동시 제거 등 자유로운 트랙 조합
- **공통음 유지 안내** — 코드 전환 시 공통음 자동 감지 및 표시
- **빠른 프리셋** — 건반만, 드럼만, 건반+드럼 등 원클릭 조합

## 🎺 EAIM Wind Orchestra v5.5

**멀티 이조 & 한글 계이름 자동 생성 시스템**

악기마다 다른 조성을 실시간으로 자동 이조하여 한글 계이름과 핑거링을 안내합니다.

### 지원 악기
| 악기 | 조성 |
|------|------|
| 🎹 플루트 / 건반 | C 악기 |
| 🎺 트럼펫 / 클라리넷 | Bb 악기 |
| 🎷 알토 색소폰 | Eb 악기 |
| 🎵 프렌치 호른 | F 악기 |

### 주요 기능
- **실시간 이조** — 원곡 C키 기준으로 각 악기 조성 자동 변환
- **한글 계이름** — 도레미파솔라시 기반 직관적 안내
- **핑거링 시각화** — 6개 키홀 실시간 점등 표시
- **합주 시뮬레이션** — 실시간 악보 진행 및 트랙 믹서
- **한글 가이드북 인쇄** — 인쇄 최적화 레이아웃

---

## 🚀 Key Features

- **멀티파트 선택** — 복수 악기 동시 선택, 원하는 조합으로 연습
- **단계별 드럼 진입** — Lv.1(킥) → Lv.2(+스네어) → Lv.3(+하이햇) → Lv.4(+심벌)
- **건반 양손 분리** — 왼손/오른손 독립 안내
- **사운드 미리듣기** — Web Audio API 기반 악기별 음색 재생
- **PWA 지원** — 설치 없이 브라우저에서 바로 실행
- **단시간 밴드 합주** — 악기별 역할 분담으로 누구나 빠르게 합주 실현
